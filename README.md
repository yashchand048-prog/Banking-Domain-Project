# 📊 Banking Portfolio Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Banking Portfolio Analysis Dashboard** built using **Power BI**. The dashboard transforms raw banking data into actionable business insights, helping stakeholders monitor loan performance, customer portfolio, risk exposure, and financial KPIs.

The objective is to provide a centralized view of banking operations, enabling data-driven decision-making through dynamic visualizations and performance metrics.

---

## 🎯 Business Objectives

* Analyze the overall banking portfolio performance.
* Monitor loan distribution across customer segments.
* Track deposits, loans, and account balances.
* Identify high-risk and low-performing portfolios.
* Enable interactive analysis using filters and slicers.
